# Terrain Rental Management System

A Spring Boot 3 application for managing terrain rentals.

## Project Details
- **Group**: com.exam
- **Artifact**: terrain-rental
- **Java Version**: 21
- **Spring Boot Version**: 3.2.0
- **Database**: MySQL

## Technologies Used
- Spring Web
- Spring Data JPA
- Validation
- Lombok
- MySQL Driver
- Spring Boot DevTools
- Spring Boot Test

## Database Configuration
1. Ensure MySQL is running.
2. Create database: `CREATE DATABASE terrain_rental;`
3. Credentials used in `application.properties`:
    - **URL**: `jdbc:mysql://localhost:3306/terrain_rental`
    - **Username**: root
    - **Password**: 123

## Setup Instructions
1. Clone the repository.
2. Run `mvn clean install`.
3. Start the application using `mvn spring-boot:run`.

## SQL Schema
The schema is automatically generated by Hibernate. Key tables:
- `terrains`: Main terrain info.
- `terrain_images`: Links images to terrains.
- `bookings`: Reservation details.
- `payments`: Financial records.
- `reviews`: User ratings.
- `favorites`: Wishlist items.

## Postman Examples

### Terrains
- **GET All**: `GET http://localhost:8080/api/terrains`
- **POST Create**: `POST http://localhost:8080/api/terrains`
  ```json
  {
    "ownerId": 1,
    "title": "New Meadow",
    "location": "Napa Valley",
    "areaSize": 500.0,
    "pricePerDay": 200.0,
    "isAvailable": true
  }
  ```

### Bookings
- **GET All**: `GET http://localhost:8080/api/bookings`
- **POST Create**: `POST http://localhost:8080/api/bookings`
  ```json
  {
    "terrain": {"id": 1},
    "renterId": 5,
    "startDate": "2026-07-01T10:00:00",
    "endDate": "2026-07-05T10:00:00",
    "totalPrice": 800.0,
    "status": "PENDING"
  }
  ```
